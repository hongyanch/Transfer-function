EMTransfer 程序使用指南
========================================

1. 文件夹内容说明
------------------
- ├── EMTransfer.exe            # 主程序（需配合MATLAB Runtime运行）
- ├── SEC_Files/                # 示例数据目录
  │   ├── 计算的参考台X.sec     # 用于“台站间传递函数计算”选项卡
  │   ├── 观测的目标台Z.sec     # 用于“局部信号提取”选项卡
  │   └── ...（共10个文件）
- ├── VC_redist.x64.exe         # Microsoft Visual C++运行库安装包（必备）
- ├── ndp462-kb3151802-web.exe  # .NET Framework 4.6.2（仅Windows 10以下系统需要）
- ├── Manual.pdf                # 程序详细操作手册（含截图步骤）
- └── Algorithm_Reference.pdf   # 算法原理论文（供学术参考）

2. 环境配置要求
------------------
■ 必需组件：
  - MATLAB Runtime R2019b (9.7) 下载链接：
    https://ww2.mathworks.cn/products/compiler/matlab-runtime.html
  - Microsoft Visual C++ Redistributable（运行VC_redist.x64.exe安装）

■ 可选组件：
  - .NET Framework 4.6+（仅旧版Windows需手动安装ndp462-kb3151802-web.exe）

3. 快速开始
------------------
1) 依次安装：MATLAB Runtime → VC_redist.x64 → .NET Framework（如需要）。  
2) 双击 EMTransfer.exe 启动程序。  
3) 通过界面加载 SEC_Files 中的数据文件，按选项卡指引操作。  

4. 注意事项
------------------
⚠ 首次运行时建议关闭杀毒软件（避免误拦截）。  
⚠ 程序所有功能均通过界面完成，无需代码输入。
